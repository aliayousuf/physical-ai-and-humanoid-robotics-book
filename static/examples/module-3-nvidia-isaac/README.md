# Module 3: NVIDIA Isaac Example Packages

This directory contains downloadable example packages for the NVIDIA Isaac module.

## Available Examples:

1. perception-pipeline-examples/
   - Complete perception pipeline setup with Isaac Sim
   - Sample sensor configurations
   - Example ROS 2 bridge configurations

2. navigation-setup-examples/
   - Nav2 configuration files for Isaac
   - Map files and parameters
   - Sample navigation scenarios

3. rl-training-examples/
   - Reinforcement learning training configurations
   - Sample environments
   - Training scripts and parameters

4. vslam-implementation-examples/
   - Visual SLAM implementation files
   - Camera calibration examples
   - Pose estimation scripts

5. path-planning-demo-examples/
   - Path planning algorithm implementations
   - Sample maps and obstacles
   - Visualization tools

6. rl-walk-cycle-examples/
   - Reinforcement learning walk cycle implementations
   - Simulation environments
   - Training configurations
