# Accessibility Statement for Physical AI & Humanoid Robotics Book

## Our Commitment to Accessibility

We are committed to ensuring digital accessibility for all learners, including those with disabilities. Our goal is to provide an inclusive learning experience that follows the Web Content Accessibility Guidelines (WCAG) 2.1 AA standards.

## Measures to Support Accessibility

We take the following measures to ensure accessibility of the Physical AI & Humanoid Robotics Book:
- Include features that allow screen reader navigation
- Ensure adequate color contrast ratios
- Provide alternative text for images and visual content
- Use semantic HTML elements appropriately
- Implement keyboard navigation for interactive components
- Ensure all functionality is available without time limits where appropriate

## Conformance Status

The content on this website is compliant with the Web Content Accessibility Guidelines 2.1 AA standard.

## Feedback

We welcome feedback on the accessibility of this resource. If you encounter any accessibility barriers or have suggestions for improvement, please contact us.

## Compatibility with Browsers and Assistive Technology

This website is compatible with:
- Modern web browsers (Chrome, Firefox, Safari, Edge)
- Screen readers (JAWS, NVDA, VoiceOver)
- Keyboard navigation

## Additional Features

This website includes features such as:
- Skip links to bypass repetitive navigation
- Sufficient color contrast for readability
- Resizable text up to 200% without assistive technology
- Clear headings and labels for navigation
- Alternative text for images and visual content
