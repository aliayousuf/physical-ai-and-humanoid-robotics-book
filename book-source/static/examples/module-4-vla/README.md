# Module 4: VLA Example Packages

This directory contains downloadable example packages for the Vision-Language-Action (VLA) module.

## Available Examples:

1. voice-to-action-examples/
   - Voice command processing pipelines
   - Speech-to-text integration
   - Action mapping implementations

2. object-detection-integration-examples/
   - Vision model integration examples
   - Object detection pipelines
   - Multi-modal fusion examples

3. manipulation-control-examples/
   - Manipulation control algorithms
   - Grasping and manipulation examples
   - End-effector control implementations

4. llm-cognitive-planning-examples/
   - LLM integration examples
   - Planning algorithm implementations
   - Reasoning and decision-making examples

5. multi-modal-fusion-examples/
   - Vision-language fusion implementations
   - Sensor data integration
   - Cross-modal processing examples

6. capstone-project-examples/
   - Complete capstone project implementations
   - Integration examples for all modules
   - Full system demonstrations
