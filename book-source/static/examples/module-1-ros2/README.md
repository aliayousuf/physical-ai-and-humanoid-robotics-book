# Module 1: ROS 2 Example Packages

This directory contains downloadable example packages for the ROS 2 module.

## Available Examples:

1. creating-first-package-examples/
   - Complete ROS 2 package structure
   - Sample C++ and Python nodes
   - Package.xml and CMakeLists.txt templates

2. launch-files-examples/
   - Various launch file configurations
   - YAML parameter files
   - Multi-node launch examples

3. rclpy-integration-examples/
   - Python ROS client library examples
   - Publisher and subscriber implementations
   - Service and action examples

4. basic-ros2-node-examples/
   - Simple publisher/subscriber examples
   - Parameter handling examples
   - Lifecycle node examples

5. urdf-robot-definition-examples/
   - Complete URDF robot models
   - Joint and link configurations
   - Material and visual properties

6. python-control-loop-examples/
   - Control loop implementations
   - Real-time control examples
   - Rate control and timing examples
